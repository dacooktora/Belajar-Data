# 📊 Data Analyst Learning

Platform pembelajaran Data Analyst dari nol sampai siap kerja dalam 6 bulan.

## 🚀 Deskripsi

Program belajar super intensif 6 bulan dengan 180 hari materi, 10 jam/hari, dan 6 project nyata. Langsung siap melamar kerja!

## ✨ Fitur

- 📚 **Materi Super Lengkap** - 180 hari belajar dengan 10 jam/hari
- 🎯 **Project Nyata** - 6 project end-to-end untuk portofolio
- 📊 **Progress Tracking** - Dashboard interaktif untuk pantau perkembangan
- 🔥 **Streak & Gamification** - Tetap termotivasi dengan streak dan badge
- 💡 **Insight & Analytics** - Analisis kelemahan dan rekomendasi belajar
- 🎓 **Siap Kerja** - Langsung siap melamar sebagai Data Analyst

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 3
- **Language**: TypeScript
- **State Management**: React Hooks + LocalStorage
- **Animations**: Framer Motion
- **Charts**: Recharts
- **UI Components**: shadcn/ui

## 🚀 Cara Menjalankan

### Prasyarat
- Node.js 18+
- npm / yarn / pnpm

### Instalasi

```bash
# Clone repository
git clone https://github.com/yourusername/data-analyst-learning.git
cd data-analyst-learning

# Install dependencies
npm install

# Jalankan development server
npm run dev
```

Buka http://localhost:3000

📦 Build untuk Production

```bash
# Build
npm run build

# Start production server
npm run start

# Atau export static
npm run export
```
